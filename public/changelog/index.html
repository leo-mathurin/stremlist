Changelog

A history of updates and improvements to the Stremlist IMDb Watchlist addon.

v1.2.1 May 14, 2025
  • Feature Added "Date Added" sorting options to display watchlist items in the order they were added to IMDb (oldest first or newest first).
  • Enhancement Changed default sorting to "Date Added (Oldest First)" which preserves the natural order from IMDb.
v1.2.0 May 12, 2025
  • Feature Added comprehensive sorting functionality: users can now sort their watchlists by title (A-Z/Z-A), year (newest/oldest first), or rating (highest/lowest first). Sort options are available both during initial setup and in configuration, with preferences persisted consistently across installations.
v1.1.1 March 28, 2024
  • Enhancement Added specific error message for private IMDb watchlists
  • Bugfix Improved error messaging to provide clearer guidance when issues occur
v1.1.0 March 26, 2024
  • Performance Replaced HTML scraping with direct GraphQL API calls to improve reliability and efficiency
  • Bugfix Fixed issue where posters and titles were displayed in French instead of English
  • Enhancement Increased the watchlist item limit from 250 to 10,000 items
v1.0.0 March 19, 2024
  • Enhancement Initial release of Stremlist
  • Feature Connect IMDb watchlist to Stremio through user ID
  • Feature Automatic refresh of watchlist data every 6-12 hours
  • Feature Support for movies and TV series from IMDb watchlists
